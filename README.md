# Public_Analysis
This project is data analysis using Public open data.  
Datasets are from '공공데이터포털' which open public datasets to people.  
[공공데이터 포털] (https://www.data.go.kr/)

## Overview
Today, there are so many datasets in our society. this is the reason why we live in 'Bigdata centry'.  
But, there are also so many datasets throw away. So, I can get insight from raw datasets existed in our government data potal. 
By doing analysis, I can get acknowledge of data analysis and visualization.

## Data imformation
1. Commercial_Imformation_Gwangju : Commercial imformation of Gwangju using folium

2. DaeJeon_KB_Card Usage : KB_Card usage in 2019, 2020 

3. InCheon_School Satisfaction : Satisfaction by Education, Sex, ... in InCheon School

4. Seoul_Covid_19 : Transition of COVID-19 Virus in 2020 at Seoul
